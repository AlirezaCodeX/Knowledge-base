# tmux

**Terminal MultiPlexer**

change pane size  
`ctrl+b ctrl+arrowKeys`

zoom a pane  
`ctrl+b z`

zoom out  
`ctrl+b z`

create a window  
`ctrl+b c`

change windows  
`ctrl+b 0 / 1`

change a window's name  
`ctrl+b ,`

time  
`ctrl+b t`

`ctrl+b page up / down` || `q` / `esc` **to quit**

## Session

create a new session  
`tmux new -s <sessionName>`

rename a session  
`ctrl+b $`

show all the sessions  
`tmux ls`

change session  
`ctrl+b (:`

Kill/terminate all Tmux sessions  
`tmux kill-server`
