#tmux
**Terminal MultiPlexer**

_ctrl+b ctrl+arrowKeys
_=> change pane size
_ctrl+b z
_=> zoom a pane || ctrl+b z => to zoom out
_ctrl+b c
_=> create a window
_ctrl+b 0 / 1
_=> change windows
_ctrl+b ,
_=> change a window's name
ctrl+b t
\*time
ctrl+b page up / down || q / esc to quit

##Session
tmux new -s <sessionName>
\*create a new session

ctrl+b $
\*rename a session

tmux ls
\*show all the sessions
