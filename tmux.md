#tmux
**Terminal MultiPlexer**

ctrl+b ctrl+arrowKeys
*change pane size
ctrl+b z
*zoom a pane || ctrl+b z => to zoom out
ctrl+b c
*create a window
ctrl+b 0 / 1
*change windows
ctrl+b ,
*change a window's name
ctrl+b t
*time
ctrl+b page up / down || q / esc to quit

##Session
tmux new -s <sessionName>
\*create a new session

ctrl+b $
\*rename a session

tmux ls
\*show all the sessions
